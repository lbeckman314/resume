\begin{itemize}[label=$\triangleright$]
    \item Updating API calls and data processing of a remote NLP document parser (\link{https://github.com/clulab/reach}{REACH}).
    \item Implementing a custom class to flag pathway and reaction revisions between varying data sets.
    \item Adding new features based on communication with users and database curators.
    \item Coauthored project updates in \emph{Nucleic Acids Research} --- \link{https://academic.oup.com/nar/article/50/D1/D687/6426058}{\emph{}The reactome pathway knowledgebase 2022}.
\end{itemize}
