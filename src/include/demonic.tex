\showoff
{\link{https://liambeckman.com/code/demonic}{demonic}}
{Interactive demos of programs and programming languages.}
{Node.js, JavaScript}
{2018 --- Present}

\begin{itemize}[label=$\triangleright$]
    \item A suite of applications that allows users to try out programs and programming languages by accessing a \link{https://liambeckman.com/code/demonic\#demonic}{web terminal} or running \link{https://liambeckman.com/code/demonic/docs/}{code snippets}.
    \item Processes are communicated via WebSockets and run in a lightweight Linux sandbox (Debian chroot secured with \link{https://firejail.wordpress.com/}{Firejail}).
\end{itemize}
