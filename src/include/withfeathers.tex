{\link{https://withfeathers.liambeckman.com}{withfeathers\textcolor{my-red}{*}}}
{Poetry web app and shell program. ```Hope' is the thing with feathers - ...''}
{Python, Flask}
{2018 --- Present}

A web app and CLI that fetches, parses, and selects a random poem by Emily Dickinson from \link{https://www.gutenberg.org/ebooks/12242}{Project Gutenberg}. A hostable web interface at \link{https://withfeathers.liambeckman.com}{withfeathers.liambeckman.com} displays a new poem every day.
