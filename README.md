To compile the tex file into a pdf, run:

```sh
pdflatex resume-liam-beckman.tex
```

It will output my resume as a PDF.

Similarly,

```sh
pdflatex coverletter-liam-beckman.tex
```

will outout a coverletter example as a PDF.
