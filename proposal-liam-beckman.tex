\myTitle{PROPOSAL}

Xena’s capabilities to collect and visualize genomic data makes it an invaluable service to scientists and healthprofessionals. Keeping the public data provided by the National Cancer Institute’s Genomic Data Commons ascurrent as possible is an important part of Xena’s functionality.

My proposed contributions would include:

\begin{itemize}
    \myItem{Identifying and updating outdated API components} The repository at gdc.xenahubs.net contains awealth of information, but some portions may be reliant on outdated data. Running a comparison betweenexisting data and data from the current GDC API would allow for the update of the database.

    \myItem{Streamlining input data} By condensing the data to include only that provided by the GDC API (therebyexcluding XML data), the existing database can more easily be maintained in the future.

    \myItem{Update new data types} As new types of data are already available from both the FM and TARGETprograms, these can be imported into the existing Xena database via the streamlined API inputs.
\end{itemize}
